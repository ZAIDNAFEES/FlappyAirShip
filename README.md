# FlappyAirShip


A 2D arcade-style flying game inspired by the classic Flappy Bird gameplay.

In Flappy Airship, players control a flying airship and navigate through challenging obstacles while trying to achieve the highest score possible.

#Features

Smooth flying mechanics  
Airship-based gameplay  
Score tracking system  
Collision detection  
Increasing challenge & addictive gameplay  
Custom made visuals and game design

# Built With

- Python Using pygame
#Goal

Survive as long as possible, avoid obstacles, and create your highest score lol :)
